Q.1] Write PHP script to read above created “book.xml” file into simpleXML object. Display attributes and elements .(Hint L simple_xml_load_file() function)

b1.php <?php $xml=simplexml_load_file("book.xml"); $list = '<h2 align="center">Books List</h2>'; $list .='<table border="3" align="center" cellpadding="5"> <tr> <th>#</th> <th>bookno</th> <th>bookname</th> <th>authorname</th> <th>price</th> <th>year</th> </tr>'; $key=1; foreach($xml as $info): $list .= "<tr> <td>".$key++."</td> <td>".$info->bookno."</td> <td>".$info->bookname."</td> <td>".$info->authorname."</td> <td>".$info->price."</td> <td>".$info->year."</td> </tr>"; endforeach; $list .= '</table>'; echo $list; ?>
