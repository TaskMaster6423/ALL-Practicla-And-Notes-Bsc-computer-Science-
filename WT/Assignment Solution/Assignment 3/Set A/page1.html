Q.1] Write a javascript to display message ‘Exams are near, have you started preparing for?’ using alert, prompt and confirm boxes. Accept proper input from user and display messages accordingly.

b1.html <html> <head> <script type="text/javascript"> function myfunc() { alert("Exams are near, have you started preparing for?"); var res=prompt("please enter your response:"); if(res!=null && res!="") { confirm("your response is "+res+"Thank you for your response!"); } else { confirm("you did not enter a response.please try again later."); } } </script> </head> <body> <input type="button" onclick="myfunc()" value="Display a confirm box"> </body> </html>
