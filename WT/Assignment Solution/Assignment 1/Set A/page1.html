Q.1] Write a PHP script to keep track of number of times the web page has been access. [Use session and cookies]

By session a1.php <html> <head> <title>Number of times the web page has been visited</title> </head> <body> <?php session_start(); if(isset($_SESSION['count'])) $_SESSION['count']=$_SESSION['count']+1; else $_SESSION['count']=1; echo"<h3>This page is accessed</h3>".$_SESSION['count']; ?> </body> </html> By cookies a1c.php <?php if(isset($_COOKIE['cnt'])) { $x=$_COOKIE['cnt']; $x=$x+1; setcookie('cnt',$x); } else { setcookie('cnt',2); echo "you accessed this page 1st time"; } echo "you accessed this page $x times"; ?>
